## 🚀 Tech Stack

### 🧩 Frontend
- React + Vite ⚡
- Tailwind CSS
- React Router v6
- Toastify
- LocalStorage for client-side state
- Framer Motion & CSS Animations

### 🔧 Backend
- Node.js + Express
- MongoDB + Mongoose
- bcryptjs for hashing
- JWT Authentication
- Multer + Sharp (image uploads & cropping)
- Postman for API testing
